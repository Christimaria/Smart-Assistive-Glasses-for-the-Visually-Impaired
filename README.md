# 👓 Smart Assistive Glasses for the Visually Impaired

### "Eyes for the Unseen — Bridging Technology and Accessibility."

This project is an AI-integrated hardware innovation designed to provide independent mobility and safety for visually impaired individuals. By combining **IoT**, **Computer Vision**, and **Real-time Sensors**, these glasses act as a smart assistant for the wearer.

---

* **🥇 I2U Funding:** Secured **Phase-II Funding** for research and prototype development.
* **📄 Published Research:** Published in the **International Journal of Engineering Research and Applications (IJERA)**, 2026.

---

## 💡 The Problem
Visually impaired individuals face significant challenges in navigating unfamiliar environments, specifically:
* Undetected obstacles at head or chest level.
* Lack of real-time visual context.
* Difficulty in independent mobility without physical assistance.

## 🚀 The Solution
Our smart glasses provide a hands-free, intuitive experience through:

### 1. Object & Obstacle Detection 📡
* Uses **Ultrasonic Sensors** to detect obstacles in the user's path.
* **Haptic/Audio Feedback:** Provides immediate alerts to prevent collisions.

### 2. AI-Integrated Vision 🤖
* Features an **ESP32-CAM** to capture and process visual data.
* **Bluetooth Audio Feedback:** Narrates the environment or obstacles directly to the user via a headset.

### 3. Real-Time Processing ⚡
* Low-latency communication between the sensors and the output device ensures the user gets information in real-time.

---

## 🛠️ Tech Stack & Components
* **Hardware:** ESP32-CAM, Ultrasonic Sensors, Arduino/Microcontrollers, Bluetooth Module.
* **Software/Languages:** C++ (Arduino IDE), Python (AI/Image Processing).
* **Communication:** Bluetooth Serial, IoT Protocols.

---

## 🗺️ Key Features
* **[✔] 360° Safety:** Multi-directional obstacle sensing.
* **[✔] Smart Narration:** Descriptive audio feedback of the surroundings.
* **[✔] Lightweight Design:** Focused on ergonomics for daily wear.
* **[✔] Long Battery Life:** Optimized for low power consumption using ESP32.

---

## 🎨 UI/UX & Hardware Design
The design focuses on **Accessibility-First** principles:
* **Minimalist Controls:** No complex buttons; focus on voice and automated triggers.
* **Ergonomic Frame:** Designed to be as close to regular eyewear as possible.
